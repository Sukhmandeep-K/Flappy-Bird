# Flappy Bird Clone

A classic arcade game built from scratch using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project focuses on game loop mechanics, physics-based movement, and procedural obstacle generation.

## 🎮 How to Play
* **Jump:** Press `Space`, `ArrowUp`, or the `X` key to flap.
* **Goal:** Fly through the gaps between the pipes to earn points.
* **Game Over:** The game ends if you hit a pipe or fall off the screen.

## 🛠️ Technical Features
* **Physics Engine:** Custom gravity and velocity logic for realistic movement.
* **Procedural Generation:** Pipes are randomly generated every 1.5 seconds to provide a unique challenge.
* **AABB Collision:** Precise hit detection between the bird and the pipes.
* **Canvas API:** High-performance 2D rendering at 60FPS.

## 🚀 Technologies Used
* **HTML5:** Canvas structure.
* **CSS3:** Game styling and layout.
* **Vanilla JavaScript:** Core game logic and engine.
